# EP06 — 純電 vs 油電混合：你的飛機是百米衝刺還是馬拉松？

(點選觀看影片)
[![影片標題](https://img.youtube.com/vi/-C_SjzHB8to/maxresdefault.jpg)](https://www.youtube.com/watch?v=-C_SjzHB8to)

大家好，我是阿仁。無人機飛不飛得起，通常是技術問題；但無人機能不能順利完成任務回來，往往是動力問題。這一集我們從系統整合者的角度，聊無人機的心臟與肌肉——動力次系統，告訴大家為什麼「選對動力，產品在標案中其實就已經安心了一半」。

## 本集重點

### 1. 任務定義動力——沒有萬能的動力系統
百米衝刺選純電（馬達安靜、反應快、震動小），馬拉松選燃油或油電（長滯空、長距離）。問自己一句：「這台飛機是要做百米衝刺，還是要跑馬拉松？」釐清任務屬性，動力選擇方向就清晰了——這不是技術高低的問題，是任務選擇的問題。

### 2. 正視能量密度——物理定律擺在那裡
鋰聚合物電池 200–250 Wh/kg，汽油 12,000 Wh/kg，差距數十倍。即使扣掉內燃機 20–30% 熱效率，燃油的有效能量仍遠高於電池。這就是為什麼高階軍用無人機還大量用內燃機。油電混合（類似 X-Trail e-Power）讓你享受燃油能量密度，又保留馬達的精準控制——在電池革命前，是長航時重酬載的解。

### 3. 擁抱由上而下設計——別再 Trial and Error
傳統流程是憑感覺買零件、試飛、過熱、換零件，倉庫堆滿燒壞的 ESC。專業做法是：先算 MTOW、推重比、單軸推力，再直接找 T-Motor、Hobbywing 這類供應商買「動力總成」——他們已經幫你把馬達、ESC、螺旋槳、散熱匹配好了。把工程師的時間留給任務整合，不要浪費在排列組合。
