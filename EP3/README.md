大家好，我是阿仁。當年我做 500 億標案的時候，軍方丟給我們一份 BOM 表，冷冷地列了四欄：元件、模組、次系統、系統。當下我背脊一涼——這其實是軍方在用一張表，無聲地告訴台灣無人機產業一個「生存秘密」：**術業有專攻**。這一集，我們像拆電腦一樣，把這台高科技玩具拆解開來，讓你看到系統工程真正的樣子。

## 本集重點

### 1. 系統商不是萬能超人，是大腦
就像你組電競電腦不會自己拿烙鐵焊主機板、自己挖矽礦做晶片。但台灣很多系統商因為找不到可靠的隊友，被迫自己繞馬達、自己焊飛控板、自己寫底層協定，結果累死、超貴又不穩定。系統商的價值，應該是「任務規劃與整合」，不是去跟一堆螺絲與電線搏鬥。

### 2. 一台無人機，就是元件→模組→次系統→系統的層層堆疊
理解這個層級結構，你才看得懂為什麼專業分工才能活下來。系統商背後必須有強大的次系統供應商：有人專做動力、有人專做通訊、有人專做機身結構。

### 3. 挑對「次系統」比買最貴的零件還重要
無人機是短板效應最明顯的產品。再貴的元件，配在不匹配的次系統裡，最後做出來的就是一台貴又難用的拼裝車。下一集開始我們會深入每一個次系統，告訴你怎麼挑、怎麼配。
