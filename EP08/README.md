# EP08 — 無人機最昂貴的眼睛：GSD、NIIRS、DRI 到底差在哪？

(點選觀看影片)
[![影片標題](https://img.youtube.com/vi/XXg1vy3dvlE/maxresdefault.jpg)](https://www.youtube.com/watch?v=XXg1vy3dvlE)

大家好，我是阿仁。99% 的無人機任務，目的只有一個：**看見**。我用過要價近 400 萬台幣的光電球，也用過幾千元的消費級雲台。光電酬載絕對不是「相機」兩個字這麼簡單——選對了，無人機是千里眼；選錯了，它就是飛在天上的近視眼。這一集教你看穿規格書的行銷話術。

## 本集重點

### 1. GSD、NIIRS、DRI 不要混在一起
**GSD** 是物理層——一個 pixel 對應地面多大；**NIIRS** 是判讀層——人能不能根據這張影像做正確判斷；**DRI**（Detect/Recognize/Identify）是任務層——整套系統在特定條件下能在多遠完成什麼等級任務。GSD 是材料、NIIRS 是判讀結果、DRI 才是任務能不能交差的答案。

### 2. 穩定性是所有影像能力的前提
拿全世界最好的望遠鏡站在劇烈震動的卡車上，你看得清楚遠方嗎？無人機本身就是強烈振動源，雲台穩定不夠，GSD 再小、NIIRS 再高都只存在規格書上。專業光電球穩定精度可以做到 0.01 度甚至更低——這才是它賣到幾百萬的真正原因。

### 3. 三光是工具箱，不是炫技；任務決定選型
EO 日間判讀、IR 夜間或複雜環境、LRF 精準定位與火控等級。不要再寫「我要 30 倍變焦」，先問自己：「要在什麼高度飛？要看什麼？我要的是發現、判斷、還是確認？」沒有一顆全能的光電球，只有最適合任務的眼睛。
