# EP01 — 無人機不是會飛的玩具，是一整套任務方案

(點選觀看影片)
[![影片標題](https://img.youtube.com/vi/p4yGV7PqZ60/maxresdefault.jpg)](https://www.youtube.com/watch?v=p4yGV7PqZ60)

大家好，我是阿仁。「無人機不就是大型遙控飛機加上一台飛控電腦嗎？」這句話我聽過太多次。這一集，我用 20 年產業經驗、240 次以上的政府委託任務經驗，跟大家打開無人機系統的第一扇門：為什麼台灣突然湧入大量人才與資金後，整個產業還在用不同的語言溝通？為什麼這套 12 集系列的使命，就是要建立「無人機系統建構的共通語言」。

## 本集重點

### 1. 無人機不是硬體，是任務體系
甲乙丙丁戊型無人機，不是遙控飛機，也不是空拍機，更不是 3C 產品。它們背後是完整的需求推導、次系統選型、飛控設計、通訊架構，還有最重要的——任務執行與後勤支援。

### 2. 無人機開發是一個完整生命週期
會飛，不代表能出任務；能出任務，不代表能量產；能量產，不代表能過驗證；能過驗證，不代表能在戰場或實務任務裡活下來。每一個環節都是專業，缺一不可。

### 3. 這套課程是為了三種人準備的
剛進入無人機領域的工程師、學生、轉職者；背負專案壓力的公司團隊、PM、技術主管；以及老闆、投資人、負責採購的人。看完之後，你會從「看不懂無人機」，變成「可以判斷無人機騙局」。

## 致謝

僅以這套無人機系統開發 12 講專輯，紀念我的恩師——**國立成功大學 航空太空工程學系 講座教授 蕭飛賓 老師**。老師的研究與教學，啟發了台灣無人機系統開發的第一代核心人才。這套內容也包含著對老師的敬意。
